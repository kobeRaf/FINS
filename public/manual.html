FINS (Financial and Networking System) - User Manual

1. Getting Started

  • Open your browser and navigate to the system's login page.
  • Enter your username and password.
  • Click the Login button to access the dashboard.

2. Navigation Overview

The left sidebar contains the main navigation menu. Click a section to expand it:

  • Dashboard - Overview of the system activity.
  • Treasury â–¾
    • Disbursements - Manage fund releases.
    • Cash Advances - View or record cash advances.
    • Collections - Record and review collections.
  • Reports - Generate financial or transaction reports.
  • Settings - Change system appearance and other configurations.

3. Dashboard

Once logged in, you will land on the Dashboard. Here you can:

  • See system summaries (e.g., pending disbursements, cash balances).
  • Access quick links to key modules.

4. Using the Treasury Module

Click Treasury to expand options:

4.1 Disbursements

  • Click Disbursements.
  • View the list of released and pending disbursements.
  • Click Add New to create a new disbursement entry.

4.2 Cash Advances

  • Select Cash Advances from the menu.
  • Click New Request to file an advance.
  • View status of past requests.

4.3 Collections

  • Choose Collections from the menu.
  • Enter collection details: source, amount, and remarks.
  • Save to update records.

5. Reports

  • Select Reports from the sidebar.
  • Filter by date range, category, or department.
  • Click Generate to export as PDF or Excel.

6. Theme Settings

  • Go to Settings > System Theme.
  • Update logo, background color, and theme color.
  • Click Save to apply changes.

7. Logging Out

  • Click your profile icon or name in the top right corner.
  • Select Logout.
  • Always log out when you are done using the system.
